# 🎬 CineMatch PRO | Mood-Based IMDb Discovery

CineMatch PRO is a modern, high-performance web application designed to help users discover movies based on their current emotional state. By leveraging real-time data from the OMDb API (synced with IMDb servers), it provides a curated browsing experience with a premium, cinematic user interface.

---

## 🚀 Features

- 🎭 **Mood-Based Engine**  
  Choose how you feel (Happy, Dark, Excited, etc.) and let the AI-driven logic suggest the perfect films.

- 🔍 **Real-Time Search**  
  Instant search functionality with debounced API calls for a smooth user experience.

- 🎞️ **Dynamic Hero Section**  
  Showcases trending titles with high-quality backdrops.

- 📊 **Deep Metadata**  
  Comprehensive movie details including IMDb ratings, Metascore, plot summaries, cast lists, and box office data.

- 📱 **Responsive Design**  
  A fully fluid layout built with Tailwind CSS, optimized for mobile, tablet, and desktop viewing.

- ✨ **Glassmorphism UI**  
  A sophisticated aesthetic featuring glass-effect cards, smooth transitions, and a dark-mode first palette.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, Tailwind CSS  
- **Interactions:** Vanilla JavaScript (ES6+)  
- **Icons:** Lucide (React / Static)  
- **API:** OMDb API  
- **Fonts:** Plus Jakarta Sans
